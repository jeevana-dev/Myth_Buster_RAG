# Myth Buster AI

An AI-powered Myth vs Fact verification system built using:

- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq Llama 3.3
- Tavily Search API

## Features

- Myth vs Fact verification
- Local semantic search using FAISS
- Live web verification using Tavily
- Source citations
